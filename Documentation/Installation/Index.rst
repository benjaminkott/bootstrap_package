============
Installation
============

The latest version can be installed via `TER`_ or via composer by running

.. code-block:: bash

   composer require bk2k/bootstrap-package

in a TYPO3 v10.4+ installation.

.. _TER: https://extensions.typo3.org/extension/bootstrap_package


Recommended Apache Modules
==========================

* mod_autoindex
* mod_alias
* mod_deflate
* mod_expires
* mod_filter
* mod_mime
* mod_headers
* mod_setenvif
* mod_rewrite
